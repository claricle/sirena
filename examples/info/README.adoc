= Info Diagram Examples

== Purpose

Info diagrams display informational messages and notices in a standardized format within documentation.

== When to Use

* Information display
* Notice visualization
* Announcement formatting
* Help text display
* Status messages

== Syntax Overview

[source,mermaid]
----
info
    title Information
    showInfo
----
